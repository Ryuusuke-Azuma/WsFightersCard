# 課題: ヘルプ画面の実装

## 概要 (Title)

設定画面に「ヘルプ」または「使い方」の項目を追加し、アプリの基本的な操作方法を説明する画面を実装する。

## 機能要求 (Feature Request)

- **設定画面への項目追加**:
  - `SettingsFragment` に、"ヘルプ" または "使い方" といった新しい `Preference` を追加する。
- **ヘルプ画面の実装**:
  - 上記の項目をタップした際に、ヘルプ画面（新しい `Fragment` または `Activity`）へ遷移する。
  - ヘルプ画面には、以下の内容を分かりやすく記載する。
    - 大会やフリー対戦の登録方法 (スケジュール画面)
    - 戦績の登録・編集・削除方法 (スケジュール画面)
    - 各種グラフや統計の見方 (アナリティクス画面)
    - データのインポート/エクスポート方法 (設定画面)
