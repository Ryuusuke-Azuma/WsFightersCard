# プロジェクト概要: WsFightersCard

## 1. コンセプト

ヴァイスシュヴァルツ（Weiβ Schwarz）プレイヤーが、日々の対戦結果を簡単かつ詳細に記録・分析し、自身の成長と戦略立案を支援するためのモバイルアプリケーション。

「**モダン ＆ クリーン**」をデザインコンセプトとし、直感的な操作性と高度な分析機能を両立します。

## 3. デザイン・ブランド

- **アプリアイコン**: ノート（記録）と円グラフ（分析）を組み合わせた、アプリの本質を象徴するデザイン。
- **配色**: 
  - メインテーマ: ライト・モノトーン（白背景にチャコールグレーのアクセント）。
  - アクセントカラー: 信頼感のあるブルー。
- **UIスタイル**: Material 3 に準拠した、システムカラーと調和するクリーンなインターフェース。

## 5. 対象ユーザー

- 日々のフリー対戦やショップ大会の結果を記録したいカジュアルプレイヤー
- 大会に向けて、自身のデッキの強みや弱みをデータで分析したい競技プレイヤー
- 複数のデッキのパフォーマンスを管理したいデッキビルダー
