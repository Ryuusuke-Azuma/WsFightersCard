# 課題: ヘルプ画面の実装

## 概要 (Title)

設定画面に「ヘルプ」または「使い方」の項目を追加し、アプリの基本的な操作方法を説明する画面を実装する。

## 機能要求 (Feature Request)

- **設定画面への項目追加**:
  - `SettingsFragment` に、"ヘルプ" または "使い方" といった新しい `Preference` を追加する。
- **ヘルプ画面の実装**:
  - `ViewPager2` を使用して、ページをめくる形式でチュートリアルを表示する。
  - 画像とテキストを組み合わせて、以下の内容を説明する。
    - 大会、戦績、ファイター、デッキの追加方法
    - リストアイテムの長押しによる編集・削除
    - SNS 連携（共有機能）
    - プロフィールとデッキの登録
    - 自分設定
    - アナリティクス画面の統計の見方
    - データ移行（インポート/エクスポート）
