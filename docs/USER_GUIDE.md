# WsFightersCard ユーザーガイド

WsFightersCard は、ヴァイスシュヴァルツ（Weiβ Schwarz）プレイヤーのための戦績管理・分析アプリです。日々のショップ大会から公式大会まで、あなたの「強さ」を可視化します。

## 📅 スケジュール管理

カレンダーから、対戦データの登録や閲覧ができます。

- **大会の登録**: カレンダーで日付を選び、右下の「＋」ボタンを押して大会名と形式（シングルス/チームス）を入力します。
- **戦績の追加**: 登録した大会をタップして「Scores」タブへ移動し、「＋」ボタンで個別の試合結果を追加します。
- **編集・削除**: 各アイテムの右側にある「3点リーダー（...）」メニューから、いつでも修正や消去が可能です。

## 📊 高度な分析機能

統計グラフ（成績統計グラフ）メニューから、多角的な分析が行えます。

- **サマリー**: 
  - 自分のトータル勝率と、形式別の詳細勝利数を確認できます。
  - トリオ戦では「チームの勝率」と「自分の勝率」を重ねて表示。チームへの貢献度が一目で分かります。
- **使用デッキ別**: デッキごとの勝率をランキング形式で表示。どのデッキが今「熱い」のかを確認しましょう。
- **苦手分析**: 負けた相手のデッキ分布を表示。環境対策のヒントになります。
- **期間フィルター**: 特定の月や年間など、集計範囲を自由に変更可能です。

## 💾 データ管理（インポート・エクスポート）

設定（Settings）画面から、データのバックアップと移行が可能です。

- **一括エクスポート**: 現在の全データを CSV 形式で保存します。スマートフォンの「ダウンロード」フォルダ等に保存して、PC で開くことも可能です。
- **一括インポート**: 外部の CSV ファイルからデータを一括で取り込みます。
  - ※形式：「GAME, 大会名, 日付(yyyy/mm/dd), SINGLES/TEAMS, メモ」および「SCORE, 自分のデッキ, 相手のデッキ, WIN/LOSE, (Team戦績), メモ」

## 👤 プロフィール

自分のプレイヤー情報や、よく使うデッキの情報を管理できます。将来的に QR コードによるプロフィール交換機能が追加される予定です。
