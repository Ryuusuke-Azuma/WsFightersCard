# 画面仕様: 設定 (Settings)

アプリ全体のデータ管理と、開発者向けのオプションを提供する画面。

## 1. データ管理 (Data Management)

### 一括インポート (Batch Import)
- **機能**: CSVファイルから大会・戦績データを一括で取り込む。
- **操作**: 
  1. 「Batch Import」をタップすると、ファイルピッカーが開く。
  2. CSVファイルを選択すると、インポートが実行される。
  3. 完了後、インポートした「大会（GAME）」の件数がトーストで表示される。
- **CSV形式**:
  - `GAME, [大会名], [日付(yyyy/MM/dd)], [形式(SINGLES/TEAMS)], [メモ]`
  - `SCORE, [自分のデッキ], [相手のデッキ], [勝敗(WIN/LOSE)], [チーム戦績(3–0など)], [メモ]`

### 一括エクスポート (Batch Export)
- **機能**: アプリ内の全データをCSV形式で書き出す。
- **操作**: 
  1. 「Batch Export」をタップすると、ファイル保存画面が開く。
  2. 保存先とファイル名を決定し、保存を実行。
  3. **初期値**: デフォルトのファイル名は `ws_fighters_card_export.csv` となる。

## 2. 開発者オプション (Developer Options)

### デバッグモード (Debug Mode)
- **機能**: ONにすると、インポート時にアセット内のサンプルデータ (`sample_import.csv`) を自動で読み込む。
- **用途**: 開発時の迅速なデータ投入と動作確認を目的とする。
