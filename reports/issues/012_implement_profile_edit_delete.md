# 課題: プロフィール編集・削除機能の実装

## 概要 (Title)

プロフィール画面に、ファイターとデッキの編集・削除機能、および「自分」を管理する機能を追加する。

## 機能要求 (Feature Request)

### 1. 編集・削除機能
- `FightersPageFragment`のリスト項目に、編集・削除メニューを追加する。
- `DecksPageFragment`のリスト項目に、編集・削除メニューを追加する。
- メニューから、対応する編集・削除ダイアログを表示する。
- 編集・削除の結果をデータベースに反映し、リスト表示を更新する。

### 2. 「自分」の管理
- ファイター追加・編集時に、「自分」として設定するためのUI（チェックボックスなど）を追加する。
- `FightersRepository`の`addFighter`および`updateFighter`メソッドが、`is_self`フラグを正しく処理するようにする。
- アプリケーション全体で、「自分」として設定されたファイターは常に1人のみとなるように制御する。
