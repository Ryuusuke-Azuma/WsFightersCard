# FightersCard アプリ概要

「FightersCard」は、カードゲーマーが日々の活動を記録し、他のプレイヤーと繋がるための総合サポートツールです。

## 1. コンセプト
カードゲーマーのためのデジタル名刺 ＆ 戦績管理デバイス。

## 2. 主なターゲット
- 自分の戦績を可視化して上達したいプレイヤー
- 大会やイベントに頻繁に参加するプレイヤー
- 他のプレイヤーと手軽に情報交換（プロフィール交換）をしたい方

## 3. 主要機能

### 📇 マイプロフィール管理
- 自分のプレイヤー名、メインデッキ、SNS連携などの情報を管理。
- 自分の「カードゲーマーとしての顔」を作成します。

### 📲 プロフィール交換 (QRコード)
- 作成したプロフィールをQRコード化。
- 相手のカメラで読み取ることで、瞬時にプレイヤー情報を交換・登録できます。

### 📝 対戦記録・スケジュール
- 対戦日、対戦相手のデッキ、勝敗結果を詳細に記録。
- 今後の大会やフリー対戦のスケジュールをカレンダー形式などで管理。

### 📈 成績の可視化 (グラフ機能)
- 過去の戦績を集計し、勝率や使用デッキごとの傾向をグラフで表示。
- 自分の得意・不得意を客観的に分析できます。

---

## 4. 技術スタック (現在の構成)
- **Kotlin Multiplatform (KMP)**: ロジックとデータ層の共通化
- **SQLDelight**: クロスプラットフォームなデータベース管理
- **Jetpack Compose / Android UI**: モダンなネイティブUI実装
- **ViewModel / LiveData**: ライフサイクルを考慮したデータ管理
