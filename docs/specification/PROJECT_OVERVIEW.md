# プロジェクト概要: WsFightersCard

## 1. コンセプト

ヴァイスシュヴァルツ（Weiβ Schwarz）プレイヤーが、日々の対戦結果を簡単かつ詳細に記録・分析し、自身の成長と戦略立案を支援するためのモバイルアプリケーション。

## 2. 主要機能一覧

各機能ごとの詳細な画面仕様は、以下のドキュメントを参照してください。

- **スケジュール管理**: [画面仕様: スケジュール](./screen_schedule.md)
- **戦績分析**: [画面仕様: 成績統計グラフ](./screen_analytics.md)
- **データ連携**: [画面仕様: 設定](./screen_settings.md)
- **プロフィール**: [画面仕様: プロフィール](./screen_profile.md)

## 3. データモデル

本アプリが扱うデータの構造（テーブル定義など）は、以下のER図を参照してください。

- **データベースER図**: [DATABASE_ER.puml](../implementation/database/DATABASE_ER.puml)

## 4. 対象ユーザー

- 日々のフリー対戦やショップ大会の結果を記録したいカジュアルプレイヤー
- 大会に向けて、自身のデッキの強みや弱みをデータで分析したい競技プレイヤー
- 複数のデッキのパフォーマンスを管理したいデッキビルダー
